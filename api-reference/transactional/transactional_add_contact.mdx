---
title: 'Quick-Add Contact'
description: 'Directly inject a contact into a specific list from your messaging infrastructure.'
---

### POST /emails/contacts

#### Request Body

<ParamField body="email" type="STRING" required>
  Recipient email address.
</ParamField>

<ParamField body="name" type="STRING" >
  Recipient full name.
</ParamField>

<ParamField body="params" type="OBJECT" >
  Custom fields / personalization metadata.
</ParamField>

<ParamField body="list_id" type="UUID" required>
  The segment ID to bind this contact to.
</ParamField>


<RequestExample>

```json Request
{
  "email": "new.user@external.com",
  "name": "Quick User",
  "list_id": "list-1234"
}
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Contact added successfully."
}
```

</ResponseExample>

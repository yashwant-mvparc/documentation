---
title: 'Message Audit Logs'
api: 'GET /emails'
description: 'Access a detailed audit trail of all developer-triggered messages and their current delivery status.'
---

### Query Parameters

<ParamField query="page" type="INTEGER"  default="1">
  Page number
</ParamField>

<ParamField query="status" type="STRING"  >
  Filter: sent, failed, bounced, complaint, opened, clicked
</ParamField>

<ParamField query="search" type="STRING"  >
  Search by email, name, or subject
</ParamField>

<ParamField query="track_id" type="STRING"  >
  Search by unique tracking ID
</ParamField>

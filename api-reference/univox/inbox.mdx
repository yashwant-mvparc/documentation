---
title: 'Message Inbox'
openapi: 'GET /inbox'
description: 'Fetch messages for a specific sending identity.'
---

### URL Parameters

<ParamField query="senderId" type="UUID">
  REQUIRED. The ID of the sender identity.
</ParamField>

<ParamField query="pageToken" type="STRING">
  Token for the next page of results.
</ParamField>

<ParamField query="label" type="STRING">
  Filter by label (e.g. INBOX, SENT). Default: inbox
</ParamField>

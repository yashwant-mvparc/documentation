---
title: 'Mark not spam'
description: 'Move message from Spam/Junk to Inbox. JSON body: senderId (UUID), messageId; mailboxId required for WildDuck and Stalwart (same as list row).'
---

### POST /inbox/messages/not-spam


<RequestExample>

```json Request
{
  "senderId": "uuid",
  "messageId": "123",
  "mailboxId": "folder path or mailbox id"
}
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "data": {
    "ok": true
  }
}
```

</ResponseExample>
